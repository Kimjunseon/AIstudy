DB(DataBase) 기본
 - 데이터의 집합
 - 여러 응용 시스템들의 [통합]된 정보들을 [저장]하여 [운영]할 수 있는 [공용] 데이터의 집합
 - 저장, 검색, 갱신, 연관, 조직화
 - 오라클, mysql 등..
 - DA(Data Architect)
 - DBA(Data Administrator)
 - DB Developer

DB 특성
 - 실시간 접근성
 - 계속적인 변화
 - 동시 공유성
 - 내용 참조
